# Finch morphological data

Individual morphological data for Darwin"s finches. finch is the all
data.frame. ind.plot.finch and sp.finch respectively correspond to the
Ilsland and the species attribute of each individual. traits.finch is
the matrix of traits with four traits in rows and 2677 individuals in
columns.

## Usage

``` r
data(finch.ind)
```

## Source

http://bioquest.org/birdd/morph.php

## Format

A data.frame of 2677 individuals in rows and 14 columns.

## Details

See http://bioquest.org/birdd/morph.php for more information on
database.

## Examples

``` r
  data(finch.ind)
```
